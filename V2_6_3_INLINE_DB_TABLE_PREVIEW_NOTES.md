# DataShield BY 2.6.3 Inline DB Table Preview

Что изменено:
- JSON в блоке загрузки заменён на inline-предпросмотр исходной базы;
- JSON в блоке маскирования заменён на inline-предпросмотр маскированной базы;
- таблицы и строки показываются прямо в основных рабочих блоках;
- технический JSON-вывод скрыт и показывается только при ошибке;
- оставлены переключатели таблиц и метаданные: таблица / столбцы / preview rows / total rows.

Что проверить:
1. загрузить SQLite базу;
2. убедиться, что прямо в блоке загрузки появилась таблица исходной БД;
3. выполнить маскирование;
4. убедиться, что прямо в блоке маскирования появилась таблица маскированной БД.
